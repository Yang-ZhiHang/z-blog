# Z-BLOG

> 本项目供学习交流使用，有问题欢迎联系 zamyang@qq.com

## 简介

绝区零是米哈游旗下的一款次元养成类游戏。

本项目 Z-BLOG 是基于 Vue3+Nuxt 的开源博客前端界面，使用 Z-BLOG 的开发者可以搭建自己的静态博客网站，或者根据自己的需要搭建后端结合使用。

文章列表中的图片取自小红书。

## 功能特点

- [x] 复刻绝区零的绳网、快速手册等样式
- [x] 文章列表支持懒加载
- [x] 支持通过配置文件统一修改网站信息

## 快速开始

1. 安装依赖

```bash
pnpm install
```

2. 启动项目

```bash
pnpm run dev
```

项目启动后进入 `localhost:3000` 即可预览
